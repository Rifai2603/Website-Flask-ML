Panduan Penggunaan Sistem

Sistem Rekomendasi Optimasi Jaringan 4G membantu Anda menganalisis data drivetest dan memberikan rekomendasi untuk meningkatkan performa jaringan.

Langkah-langkah Penggunaan

1

Upload File Data Drivetest

Unggah file data jaringan dalam format CSV yang diperoleh dari software Genex Assistant melalui halaman Upload File.

Isi form Site ID dan Alamat Site
Pilih file CSV dari komputer Anda
Klik tombol "Analisa" untuk memproses data
2

Lihat Hasil Analisis

Setelah proses analisis selesai, sistem akan menampilkan hasil berupa tabel data dan rekomendasi optimasi jaringan.

Tabel menampilkan data koordinat, parameter jaringan, dan jenis rekomendasi
Ringkasan rekomendasi optimasi ditampilkan di bagian atas hasil
3

Download Hasil Analisis

Anda dapat mengunduh hasil analisis dalam format CSV untuk digunakan lebih lanjut.

Klik tombol "Download Hasil Analisis" pada halaman hasil
File CSV akan berisi data lengkap hasil analisis
4

Lihat Detail Parameter

Untuk melihat detail parameter jaringan, kunjungi menu Parameter 4G.

Pilih parameter yang ingin dilihat (RSRP, SINR, Throughput DL, Throughput UL)
Lihat visualisasi peta, tabel, dan grafik perbandingan sebelum dan sesudah optimasi

Format File CSV

File CSV yang diunggah harus memiliki format yang sesuai dengan output dari software Genex Assistant dan harus memiliki kolom-kolom berikut:

Nama Kolom Deskripsi Satuan
Longitude Koordinat bujur lokasi pengukuran Derajat desimal
Latitude Koordinat lintang lokasi pengukuran Derajat desimal
LTE PCC Serving RSRP(dBm) Reference Signal Received Power dBm
LTE PCC Serving SINR(dB) Signal to Interference plus Noise Ratio dB
LTE DL App Throughput(kbps) Kecepatan unduh data kbps
LTE UL App Throughput(kbps) Kecepatan unggah data kbps
Penting: Pastikan nama kolom sesuai dengan format di atas. Sistem akan otomatis mengisi nilai yang kosong dengan nilai median dari data yang ada.

Interpretasi Parameter

RSRP (Reference Signal Received Power)

Mengukur kekuatan sinyal yang diterima dari bts.

Kategori Nilai (dBm) Kualitas
Sangat Bagus -85 s/d 0
Bagus -95 s/d -85
Normal -100 s/d -95
Buruk -105 s/d -100
Sangat Buruk -150 s/d -105
SINR (Signal to Interference plus Noise Ratio)

Mengukur kualitas sinyal yang diterima relatif terhadap interferensi dan noise.

Kategori Nilai (dB) Kualitas
Sangat Bagus 10 s/d 50
Bagus 5 s/d 10
Normal 0 s/d 5
Buruk -5 s/d 0
Sangat Buruk -20 s/d -5
Throughput Downlink

Mengukur kecepatan unduh data dari jaringan ke perangkat.

Kategori Nilai (Kbps) Kualitas
Sangat Bagus 20.000 s/d 300.000
Bagus 10.000 s/d 20.000
Normal 5.000 s/d 10.000
Buruk 3.000 s/d 5.000
Sangat Buruk 0 s/d 3.000
Throughput Uplink

Mengukur kecepatan unggah data dari perangkat ke jaringan.

Kategori Nilai (Kbps) Kualitas
Sangat Bagus 20.000 s/d 300.000
Bagus 10.000 s/d 20.000

Pertanyaan Umum (FAQ)

Sistem mendukung file dalam format CSV (Comma Separated Values) yang dihasilkan dari software Genex Assistant. File harus memiliki kolom-kolom yang diperlukan seperti Longitude, Latitude, RSRP, SINR, Throughput DL, dan Throughput UL.

Untuk mendapatkan file data drivetest dari Genex Assistant, ikuti langkah-langkah berikut:
  1. Buka software Genex Assistant
  2. Impor data drivetest yang telah dikumpulkan
  3. Lakukan analisa pada data drivetest yang telah diimpor
  4. Pilih menu "More" kemudian klik pada bagian "Export" pilih "IE"
  5. Pastikan semua parameter yang diperlukan (Longitude, Latitude, RSRP, SINR, Throughput Downlink, Throughput Uplink) dicentang untuk diekspor
  6. Simpan file CSV ke komputer Anda

Sistem secara otomatis akan menangani nilai yang kosong (missing values) pada data Anda. Untuk setiap parameter (RSRP, SINR, Throughput DL, Throughput UL), sistem akan menghitung nilai median dari data yang tersedia dan menggunakan nilai tersebut untuk mengisi nilai yang kosong. Namun, untuk kolom Longitude dan Latitude, baris dengan nilai kosong akan dihapus karena koordinat lokasi sangat penting untuk analisis.

Hasil rekomendasi optimasi ditampilkan dalam bentuk ringkasan dan detail per titik pengukuran:
  • Ringkasan: Menampilkan jenis rekomendasi optimasi yang paling banyak muncul untuk site yang dianalisis.
  • Detail per titik: Pada tabel hasil, kolom "Jenis Rekomendasi Optimasi" menunjukkan rekomendasi spesifik untuk setiap titik pengukuran.
  • Perbandingan parameter: Pada halaman parameter, Anda dapat melihat perbandingan nilai parameter sebelum dan setelah optimasi untuk mengevaluasi potensi peningkatan performa jaringan.

Saat ini, sistem hanya mendukung analisis satu site dalam satu waktu. Untuk menganalisis site lain, Anda perlu menyelesaikan analisis site saat ini terlebih dahulu, kemudian klik tombol "Analisa Baru" untuk memulai analisis site berikutnya. Pastikan Anda telah mengunduh hasil analisis site sebelumnya jika diperlukan, karena data akan dihapus saat memulai analisis baru.
